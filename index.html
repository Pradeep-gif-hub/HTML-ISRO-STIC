Project Methods

Heat Pulse Method

This method is used as heat transient in which it correlates the effect with change in time, it comprises three probes: the central probe is a heater and the remaining two outer probes are temperature sensors.

Double Sided Transient Plane Source Method

The Double Sided Transient Plane Source (DSTPS) method measures thermal conductivity by placing a sensor between two identical sample halves.

Solution

To calculate the thermal conductivity of materials, an apparatus comprising the following components is used:

  • Arduino UNO
  • ADC (Analog-to-Digital Converter)
  • Heater
  • Temperature Sensor
